# 통합 서비스 대시보드 - 사용자 가이드

## 📋 개요

**통합 서비스 대시보드**는 날씨, 물류, 에너지 효율 정보를 한 곳에서 관리할 수 있는 전문적인 웹 애플리케이션입니다. 건축 청사진 미학을 바탕으로 한 깔끔한 인터페이스로 기술적이고 정밀한 데이터 관리를 제공합니다.

## 🚀 시작하기

### 1. 로그인
- 웹사이트에 접속하면 로그인 페이지가 표시됩니다.
- **"로그인하여 시작하기"** 버튼을 클릭하여 Manus OAuth를 통해 인증합니다.
- 로그인 후 대시보드에 접근할 수 있습니다.

### 2. 대시보드 레이아웃

**좌측 사이드바**
- 프로젝트 로고 및 네비게이션 메뉴
- 현재 사용자 정보 표시
- 로그아웃 버튼

**상단 네비게이션 바**
- 메뉴 토글 버튼 (사이드바 열기/닫기)
- 대시보드 제목

**메인 콘텐츠 영역**
- 실시간 모니터링 섹션
- 날씨, 물류, 에너지 정보 카드

## 🌤️ 날씨 정보 관리

### 날씨 데이터 조회
1. 대시보드의 **"날씨 정보"** 섹션을 확인합니다.
2. 다음 정보가 표시됩니다:
   - **온도**: 현재 기온 및 체감 온도
   - **습도**: 공기 중 수증기 함량 (%)
   - **풍속**: 바람의 속도 (km/h)
   - **시정**: 수평 가시거리 (m)
   - **기압**: 대기압 (hPa)
   - **자외선 지수**: UV 강도 (0-11)
   - **강수량**: 예상 강수량 (mm)

### 데이터 저장
- 조회한 날씨 정보는 자동으로 데이터베이스에 저장됩니다.
- 히스토리 조회를 통해 과거 데이터를 확인할 수 있습니다.

## 📦 물류 추적

### 배송 상태 확인
1. **"물류 추적"** 섹션에서 배송 정보를 확인합니다.
2. 다음 정보가 표시됩니다:
   - **배송 번호**: 추적 번호
   - **배송 상태**: 배송중, 배송완료, 배송대기 등
   - **출발지/도착지**: 배송 경로
   - **배송사**: 물류 회사명
   - **예상 배송일**: 도착 예정일
   - **무게**: 상품 무게 (g)
   - **거리**: 배송 거리 (km)
   - **비용**: 배송료 (원)

### 배송 추적 업데이트
- 배송 정보는 실시간으로 업데이트됩니다.
- 배송 상태 변경 시 자동으로 기록됩니다.

## ⚡ 에너지 효율 모니터링

### 에너지 사용량 확인
1. **"에너지 효율"** 섹션에서 에너지 데이터를 확인합니다.
2. 다음 정보가 표시됩니다:
   - **시설명**: 에너지를 사용하는 건물/시설
   - **에너지 종류**: 전기, 가스, 수도 등
   - **사용량**: 해당 기간의 총 사용량
   - **비용**: 에너지 요금 (원)
   - **효율성**: 에너지 효율 지수 (%)
   - **탄소 배출**: CO₂ 배출량 (kg)
   - **최대 사용량**: 피크 타임의 사용량
   - **평균 사용량**: 기간 평균 사용량
   - **추세**: 사용량 증감 추세

### 효율성 개선
- 효율성 지표를 통해 에너지 사용 패턴을 분석합니다.
- 추세 정보를 바탕으로 에너지 절감 계획을 수립할 수 있습니다.

## 🎨 인터페이스 설명

### 디자인 특징
- **로얄 블루 배경**: 전문적이고 신뢰감 있는 색감
- **그리드 패턴**: 건축 청사진 스타일의 배경
- **기술 선 드로잉**: CAD 스타일의 경계선과 프레임
- **흰색 타이포그래피**: 명확한 텍스트 가독성

### 카드 컴포넌트
- 각 정보는 독립적인 카드로 표시됩니다.
- 카드는 관련 정보를 그룹화하여 시각적 계층을 제공합니다.
- 아이콘을 통해 정보 유형을 빠르게 식별할 수 있습니다.

## 📊 데이터 관리

### 데이터 저장
- 모든 조회 데이터는 자동으로 데이터베이스에 저장됩니다.
- API 호출 기록이 유지되어 사용 현황을 추적할 수 있습니다.

### 히스토리 조회
- 과거 조회한 데이터를 시간순으로 확인할 수 있습니다.
- 데이터 변화 추이를 분석할 수 있습니다.

## 🔒 보안 및 개인정보

- **Manus OAuth 인증**: 안전한 로그인 방식 사용
- **세션 관리**: 자동 로그아웃으로 보안 강화
- **데이터 암호화**: 모든 통신은 HTTPS를 통해 암호화됨
- **사용자별 데이터 격리**: 각 사용자의 데이터는 독립적으로 관리됨
